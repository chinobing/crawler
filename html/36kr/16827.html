“未来接入信息不再是问题，大学不能依靠教内容收费”，36氪对话Minerva创始人Ben Nelson

Ace Wang • 2014-11-10 • 其他
本科教学的钻石班

不知道Minerva是什么？扫盲推荐阅读：

  • 挑战常青藤盟校，创业公司 The Minerva Project 获得2500万美元的巨额种子投资打造在线版哈佛大学

  • Minerva 获得7000万美元 B 轮融资，好未来与 Benchmark 共同投资

  • 和 Kenn Ross 聊聊 Minerva 的7000万美元融资，以及本科大学的教育

上周四，突然得知Minerva创始人Ben Nelson来中国的消息，随后收到了Minerva官方的采访邀请，约国内关注Minerva大学的媒体们一起和Ben当面聊聊，于是，就有了周五发生的这一切对话：

｢幸会幸会｣，面前的高个白人说道，眼神含笑——如果只听口音，我几乎要认为这是个说闽南普通话的小哥。

｢请选一张｣，他在手里摊开自己的名片——Kenn Ross，中文名是罗凯，Minerva大学亚太区头头——嗯，一沓，每张颜色都不一样，像极了彩虹。我抽出了红色。

｢这么多？｣ 我实在很好奇。

｢人生应该多一些选择嘛｣，我诧异了一下，同他、还有周围的人都笑了起来。

｢这哥们真有意思｣，我和苑伶的兴趣一下子都被调动起来了。

很快，他将我们引荐给会议桌中央的眼镜叔——他的老大 Ben Nelson——沉静、友善、有点学院气。

｢中国深圳是我出游世界的第一站｣，他对中国似乎抱有别样的理解和情感。｢我觉得这个国家是开放的｣，他顿了顿，｢在这，有些机构和人正实践着最前沿的教育创新，敢于尝试新事物｣——Ben 表示，中国给了他启发，以及正面支持——当然，他提到了好未来在价值观上的一致，以及对 Minerva 的投资。

接下来，他给我们展示了两段视频，一短——典型的宣介片，一长——还未公布、由 Founding Class 成员参与录制，主要是关于在 Minerva 生活和学习的体验——各种肤色、口音、国籍、故乡... 其中有6位来自中国——具体的内容我肯定不能说啦，如果你在网上搜索 Minerva Project 的视频，我相信你会希望自己成为其中一员的。

｢接下来我们就做自由提问吧｣，Ben 投来自信而诚恳的目光——哟，干活了。

一、Minerva 计划的模式是否可大面积推广？是否只能服务小部分、最具潜力的年轻人——就像电影｢安德的游戏｣里那样？

Ben：当大家在思考教育模式的时候，大家都往上看，去看哈佛、耶鲁这样的学校，但是他们的问题是他们都建立在不大可行的经济体系上：包括建大量的基础设施、请专业的教授，并且在研究上花费高昂的成本，这导致他们最关注的并不是本科教育的质量，更多的是研究的成果，但本科学生却要花费高昂的学费换取不对等的教学质量。而当其他学校看向哈佛、耶鲁的时候，他们都试图去复制这种并不是真正可持续的模式。

Minerva则希望招在全球招到最顶尖的学生，他们建立全球化的体验以及人际关系的网络。四年以后，当这些学生毕业了，向社会证明了他们是非常优秀的学生，而后全世界重新思考教育的事情的时候，他们便不再看向哈佛，而是会看向Minerva，他们会看见原来有一个可以以更加低廉的成本来运行大学的方式——这个就是Minerva最大的意义。

而更低廉成本意味着更可以被复制——这种模式就可以在全球更多地方推行的——也就是说，Minerva 将会惠及越来越多的人，而不只是一小部分。

二、推广、实践时，是否发现在发达国家比发展中国家更容易？

Ben：不会。尽管各地区有不一样的特色，但没有发现这种问题。

三、扩招速度如何？

Ben：每年增幅200～250个学生——但其中单一国别的学生不会超过四分之一。

举例子：今年招200个，那么来自美国的学生不可能超过50个——这和美国现在绝大部分学校动辄90%来自本土是很不一样的。小一点规模国家的学生则差不多10个。

当招到2000人的规模时，就会保持这个规模。我们必须指出：绝对没有刻意卡这个数字——只要你满足条件，我们就有意录取你。

四、教育产业是一门特别的生意。它的品牌价值尤其倚赖于教学质量。在3个维度上，你会如何保证品质呢：产品——是否会侧重于移动体验？投资人——下一轮如何选择？如何衡量教学质量——比如使用考试体系？

Ben：

  • 产品：现如今，技术的更迭频率很高。总体而言，技术越来越好，Minerva 也会与时俱进

  • 投资人：这是自然、双向选择的过程。无论是我们，还是我们所选择的投资人，都认可品牌价值至上——给你两个选择：稍微损害品牌价值但盈利翻倍，或者巩固品牌不去赚这些钱。我们和我们的投资人会选择后者。这听上去有点违背投资人的普遍形象。但我们发现，最棒的投资人往往也最具前瞻性，比如投了我们的 Benchmark Capital

  • 教学质量考核：这就需要谈到我们的课程设计，因为它与衡量系统的结合非常紧密。说回录取，我们是有绝对标准的——这点和许多学校不一样——而我们选择 Founding Class 的标准会沿用下去，只会提高或保持，绝对不会降低。教学期间，我们每天都会考核学生在应用思维方式时的表现。此外，还有老师的教学质量——这一点使用淘汰制——绝对部分美国顶尖大学不这么做，它们以科研结果为导向

五、投资者的回报？

Ben：Minerva 的目标在于打造最优质的高等教育体系——我们的品牌价值会非常高。

试想：你有无限的钱，能买两个品牌： IBM，还是哈佛？IBM 光品牌就值650亿美元，你想想哈佛呢？

这是给投资人和其他人带来的最大回报。

六、如何看待教育创新中体现的去中心化和正在提升的开放度？前者的示例：Skillshare 这样的技能教学众包平台；后者如Coursera、EdX、Udacity 等公开课平台。在这些机构和产品身上，是否看到合作或者竞争的点？

Ben：这就谈到来 Minerva 与公开课等互联网教育创新的关系：

正是互联网教育的存在，才使得 Minerva 有了意义——Minerva 没有入门课，花几万块学｢统计101｣是很荒谬的。所以我们不做任何单纯信息内容的传授，我们不是教内容的学校，我们做的是教给学生129种思考问题的方法，配合互联网的信息，来培养学生分析问题的能力。

我们相信未来 Access to Info（接入信息）不再是个问题。在2015、2016年的时候，互联网教育的存在将会逼迫现有的传统机构做出更多改变，大学将不再能依靠教给学生免费的内容来收费。

另外，也许你会说，这129种思维方式也可以当做内容被复制——理论上这是可行，如果有其他机构希望借鉴 Minerva 的教学方式，我们也是高兴的。但是光是第一年的四门课程，要让那些非常顶尖的教授按照“教思维方式”而不是“教内容”的方式去交，就需要让他们看3000页的指导材料。

所以有具备足够知识和教学方法的教授、有能配合教学的在线平台、有定制的课程设计，这三者是环环相扣的，这些就是壁垒。

七、给教授、员工的薪资？

Ben：一句话：很有竞争力。

说细一点：高薪、停车位、医疗、保险，等等等等。

八、学费与涨学费是个困扰人类多年的问题

Ben：学费——1万美元／年，生活费和其它学校差不多——8000美元/年。

我们会保证学费的增长低于通胀增长率——传统的学校在普遍高于2～3倍。

另外，Minerva 有能力在全球范围内灵活地选择最经济可行的地区作为校园，所以能长期控制总体费用在3万美元内。

九、Minerva 给学生带来的价值与增值？

Ben：先谈谈 HBS、沃顿等商学院的增值模式——校友人脉——这其实是个非常模糊的概念。

不是说，你去某个公司面试，因为你是校友，所以你就获得一份好工作。校友人脉中最有价值的往往是同学4年、相互熟识的那部分——Minerva 的小班就保证了这一点。

另外，我们会为所有学生安排一位助手——主动在他有融资、公关、实习就业、创业需求时予以帮助，一直到这位学生退休。

话说回来，我们和许多融资机构、投行、咨询公司都有 Buddy Design 的机制——这些机构表现出强烈的意愿：愿意赞助学生的学费——对于这些 B 端 “客户” 而言，这是很划算的事——没什么比第一流的人才更吸引他们的了。
